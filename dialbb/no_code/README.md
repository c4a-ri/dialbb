## インストール方法
* whlファイルをダウンロードします:dialbb_nc-x.x.x-py3-none-any.whl  
  [GitHub](https://github.com/c4a-ri/dialbb-dev/tree/dialbb_nc/dist)
* パッケージのインストール
```sh
> pip install <ダウンロードしたwhlファイルの名前> 
```

------

## DialBB ノーコードの起動コマンド
```sh
> dialbb-nc
```
